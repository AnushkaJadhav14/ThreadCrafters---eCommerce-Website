ThreadCrafters E-Commerce Website
=================================

Project Overview:
-----------------
ThreadCrafters is a Front-End eCommerce website dedicated to selling stitched traditional attires. The website offers a variety of products categorized for men, women, boys, and girls. It features user-friendly navigation, a secure cart system, and detailed policy pages.

Folder Structure:
-----------------
ThreadCrafters/
├── images/                # Contains product images
│   ├── men/               # Images of products for men
│   ├── women/             # Images of products for women
│   ├── boys/              # Images of products for boys
│   └── girls/             # Images of products for girls
├── about.html             # About the brand and company
├── cart.html              # Shopping cart page
├── cart.js                # JavaScript for cart functionality
├── contact.html           # Contact form for inquiries
├── contact.js             # JavaScript for contact form validation
├── home.css               # Styling for the homepage
├── login.html             # User login page
├── privacy-policy.html    # Privacy policy details
├── products.js            # JavaScript for product interactions
├── readme.txt             # Project documentation
├── register.html          # User registration page
├── return-exchange-policy.html # Return and exchange policy
├── shop.css               # Styling for the shop page
├── shop.html              # Shop page showcasing all products
├── shop.js                # JavaScript for shop page functionality
├── terms.html             # Terms and conditions page

How to Run:
-----------
1. Open `home.html` in any modern web browser to access the homepage.
2. Navigate through the site using the menu to explore the About, Shop, Contact, and other sections.
3. Use the `cart.html` to view or manage your shopping cart.
4. For inquiries, use `contact.html` and ensure JavaScript is enabled for validation.
5. All styles and scripts are linked internally; ensure all files are in the correct directory structure.

Features:
---------
- **Homepage:** Overview of the brand with quick links to key sections.
- **Shop Page:** Browse products by category with detailed descriptions and images.
- **Cart Functionality:** Add, remove, and manage items in the shopping cart.
- **User Pages:** Login and register pages for user account management.
- **Contact Form:** Submit inquiries with attachment support and validation.
- **Policy Pages:** Detailed privacy, terms, and return/exchange policies for transparency.

Dependencies:
-------------
- The website uses only HTML, CSS, and JavaScript.
- Ensure the project is served from a local or web server for file dependencies to load correctly.

Notes:
------
- Make sure the `images/` folder contains all necessary product images in their respective subfolders.
- Validate all external links and file paths to ensure smooth navigation.
